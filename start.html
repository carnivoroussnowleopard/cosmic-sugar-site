01 / START SCENARIOS

처음 위치가 장르를 정합니다.

각 시작점은 같은 도시를 다른 각도에서 보게 합니다. 난이도는 정보량과 감시 압박 기준입니다.

A / LOW / Pink-1

신규 SHI 입사자

임시 ID와 첫 Candy packet을 받고 오리엔테이션을 기다립니다. 세계관 설명이 가장 자연스럽습니다.

B / MID / Sleep Pod Clinic

기억 손상 직원

이미 직원이지만 일부 기억이 비어 있습니다. 기록과 기억의 모순을 따라가는 미스터리에 강합니다.

C / HIGH / RND Sublevel

R&D 테스트 대상

시스템과 Hollowing을 빠르게 체감합니다. 초반부터 압박이 강하므로 첫 플레이에는 무겁습니다.

D / MID / Silver-2

Sugar Idols 후보생

방송, PR, 카메라 앞의 밝음과 카메라 밖의 자아 분리를 다루기 좋습니다.

E / HIGH / Cyan-3 or Black-1

Luna Cell 접촉자

저항 루트가 빠르게 열립니다. 대신 실패 비용이 크고 신뢰 검증이 까다롭습니다.

F / MID-HIGH / Cyan-1

NEU 협력자

쿠토와 중립 루트가 열립니다. 선택하지 않는 선택과 개입 타이밍이 핵심입니다.

02 / DEFAULT OPENING

선택이 없으면 신규 입사자로 시작합니다.

가장 권장되는 첫 장면은 너무 친절하고, 너무 안전하고, 너무 달콤한 Pink District입니다.

INITIAL STATE

기본 시작값

  • 장소: Pink District / Pink-1
  • 상태: 임시 ID 수령
  • 지급품: 첫 Candy packet
  • 대기: 공식 오리엔테이션
  • 분위기: 너무 친절하고, 너무 안전하고, 너무 달콤함
ONE ANOMALY

이상징후는 하나만

초반에는 모든 진실을 열지 않습니다. 복도 방송 오류, 사라진 선배 이름, 캔디 포장지의 작은 경고문처럼 작은 균열 하나가 충분합니다.

처음부터 모든 진실을 알 필요는 없습니다.
03 / SCENE RHYTHM

장면은 평범함에서 균열로 갑니다.

이 구조를 따르면 공포가 직접 설명되지 않고 상황 속에서 새어 나옵니다.

1. PLACE

장소와 표면 분위기를 먼저 제시합니다.

2. DAILY

평범한 행동 하나를 둡니다. 배급, 훈련, 방송, 식사 같은 일상입니다.

3. ANOMALY

작은 이상징후를 하나만 삽입합니다.

4. NPC

NPC는 자기 방어기제대로 반응합니다. 설명보다 회피와 말투가 우선입니다.

5. PRESSURE

캔디, Haze, 추출률, 감시 같은 시스템 압박을 조용히 반영합니다.

6. NEXT

다음 행동의 실마리만 남기고 장면을 끝냅니다.
